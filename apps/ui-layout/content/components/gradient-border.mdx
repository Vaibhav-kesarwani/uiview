## Usage

```css global.css
@property --border-angle {
  inherits: false;
  initial-value: 0deg;
  syntax: '<angle>';
}
```

```ts tailwind.config.ts
tailwind.config = {
  theme: {
    extend: {
      animation: {
        border: 'border 4s linear infinite',
      },
      keyframes: {
        border: {
          to: { '--border-angle': '360deg' },
        },
      },
    },
  },
};
```
